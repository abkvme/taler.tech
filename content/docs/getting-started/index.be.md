Вітаем у TALER! Гэта кіраўніцтва дапаможа вам пачаць працу з блакчэйнам TALER, загрузіўшы і ўсталяваўшы дадатак-кашалёк.

## Перадумовы

Перад пачаткам працы пераканайцеся, што ваша сістэма адпавядае наступным патрабаванням:

- **Аперацыйная сістэма**: Windows 10/11, macOS (Apple Silicon) або Linux (x64/ARM64)
- **Дыскавая прастора**: Не менш за 10 ГБ вольнага месца (для даных блакчэйна)
- **Інтэрнэт-злучэнне**: Стабільнае злучэнне для сінхранізацыі блакчэйна
- **ААП**: Мінімум 2 ГБ даступнай памяці

## Загрузка кашалька TALER

### Крок 1: Наведайце афіцыйную старонку рэлізаў

Загрузіце апошнюю версію кашалька TALER з афіцыйнага рэпазіторыя GitHub:

**Апошні рэліз**: [GitHub Releases](https://github.com/abkvme/taler/releases/latest)

### Крок 2: Выберыце вашу платформу

Выберыце адпаведны пакет для вашай аперацыйнай сістэмы:

| Платформа | File Pattern | Апісанне |
|----------|------|-------------|
| **Windows (64-bit)** | `taler-*-win64.zip` | Выканальны файл для Windows 10/11 |
| **macOS (M1/M2/M3+)** | `taler-*-macos-arm64.tar.gz` | Натыўная зборка для Apple Silicon |
| **Linux (64-bit)** | `taler-*-linux-x64.tar.gz` | Бінарны файл для Linux x86_64 |
| **Linux (ARM64)** | `taler-*-linux-arm64.tar.gz` | Бінарны файл для ARM64/aarch64 |



### Крок 3: Праверка загрузкі

Пасля загрузкі праверце цэласнасць файла, звераючыся з памерам файла і кантрольнай сумай SHA256 (даступныя на старонцы рэліза).

## Наступныя крокі

Пасля загрузкі кашалька TALER для вашай платформы:

1. **Усталяванне**: Прытрымлівайцеся інструкцый па ўсталяванні для канкрэтнай платформы ў кіраўніцтве [Наладка кашалька](/be/docs/wallet-setup/)
2. **Першы запуск**: Даведайцеся, як запусціць кашалёк і сінхранізавацца з блакчэйнам
3. **Стварэнне кашалька**: Кашалёк будзе створаны аўтаматычна пры першым запуску
4. **Пачатак выкарыстання**: Пачніце адпраўляць і атрымліваць манеты TALER

Для атрымання падрабязных інструкцый па ўсталяванні і наладцы перайдзіце да дакументацыі [Наладка кашалька](/be/docs/wallet-setup/).
