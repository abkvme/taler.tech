Ласкаво просимо до TALER! Цей посібник допоможе вам розпочати роботу з блокчейном TALER, завантаживши та встановивши застосунок-гаманець.

## Передумови

Перед початком роботи переконайтеся, що ваша система відповідає наступним вимогам:

- **Операційна система**: Windows 10/11, macOS (Apple Silicon) або Linux (x64/ARM64)
- **Дисковий простір**: Щонайменше 10 ГБ вільного місця (для даних блокчейну)
- **Інтернет-з'єднання**: Стабільне з'єднання для синхронізації блокчейну
- **ОЗП**: Мінімум 2 ГБ доступної пам'яті

## Завантаження гаманця TALER

### Крок 1: Відвідайте офіційну сторінку релізів

Завантажте останню версію гаманця TALER з офіційного репозиторію GitHub:

**Останній реліз**: [GitHub Releases](https://github.com/abkvme/taler/releases/latest)

### Крок 2: Виберіть вашу платформу

Оберіть відповідний пакет для вашої операційної системи:

| Платформа | File Pattern | Опис |
|----------|------|-------------|
| **Windows (64-bit)** | `taler-*-win64.zip` | Виконуваний файл для Windows 10/11 |
| **macOS (M1/M2/M3+)** | `taler-*-macos-arm64.tar.gz` | Нативна збірка для Apple Silicon |
| **Linux (64-bit)** | `taler-*-linux-x64.tar.gz` | Бінарний файл для Linux x86_64 |
| **Linux (ARM64)** | `taler-*-linux-arm64.tar.gz` | Бінарний файл для ARM64/aarch64 |



### Крок 3: Перевірка завантаження

Після завантаження перевірте цілісність файлу, звіривши розмір файлу та контрольну суму SHA256 (доступні на сторінці релізу).

## Наступні кроки

Після завантаження гаманця TALER для вашої платформи:

1. **Встановлення**: Дотримуйтесь інструкцій з встановлення для конкретної платформи в посібнику [Налаштування гаманця](/uk/docs/wallet-setup/)
2. **Перший запуск**: Дізнайтеся, як запустити гаманець і синхронізуватися з блокчейном
3. **Створення гаманця**: Гаманець буде створено автоматично при першому запуску
4. **Початок використання**: Почніть надсилати та отримувати монети TALER

Для отримання детальних інструкцій з встановлення та налаштування перейдіть до документації [Налаштування гаманця](/uk/docs/wallet-setup/).
