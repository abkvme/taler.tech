Цей посібник проведе вас через процес встановлення та налаштування гаманця TALER у вашій системі.

## Встановлення

### Встановлення в Windows

1. **Розпакуйте архів**:
   - Клацніть правою кнопкою миші на загружаний файл `taler-*-win64.zip`
   - Виберіть "Видобути все..."
   - Виберіть папку призначення (наприклад, `C:\Program Files\TALER`)

2. **Знайдіть виконуваний файл**:
   - Перейдіть до розпакованої папки
   - Знайдіть `taler-qt.exe` (застосунок GUI-гаманця)

3. **Опціонально - Створіть ярлик на робочому столі**:
   - Клацніть правою кнопкою миші на `taler-qt.exe`
   - Виберіть "Надіслати" → "Робочий стіл (створити ярлик)"

### Встановлення в macOS

1. **Розпакуйте архів**:
   ```bash
   tar -xzvf taler-*-macos-arm64.tar.gz
   ```

2. **Перемістіть до Applications** (опціонально):
   ```bash
   sudo mv taler-*/bin/* /usr/local/bin/
   ```

3. **Надайте права на виконання**:
   ```bash
   chmod +x /usr/local/bin/taler-qt
   ```

### Встановлення в Linux

1. **Розпакуйте архів**:
   ```bash
   tar -xzvf taler-*-linux-x64.tar.gz
   cd taler-*
   ```

2. **Встановіть в систему** (опціонально):
   ```bash
   sudo install -m 0755 -o root -g root -t /usr/local/bin bin/*
   ```

3. **Або запустіть з розпакованої директорії**:
   ```bash
   ./bin/taler-qt
   ```

## Запуск гаманця TALER

### Запуск через командний рядок

GUI-гаманець TALER можна запустити з командного рядка з різними опціями.

#### Базовий запуск

**Windows**:
```cmd
taler-qt.exe
```

**macOS / Linux**:
```bash
taler-qt
```

#### Запуск з користувацькою директорією даних

Вкажіть користувацьке розташування для даних блокчейну:

```bash
taler-qt -datadir=/path/to/custom/directory
```

#### Запуск у тестовій мережі

Для цілей тестування запустіть гаманець у тестовій мережі:

```bash
taler-qt -testnet
```

#### Загальні параметри командного рядка

| Опція | Опис |
|--------|-------------|
| `-datadir=<dir>` | Вказати розташування директорії даних |
| `-conf=<file>` | Вказати файл конфігурації |
| `-testnet` | Використовувати тестову мережу |
| `-prune=<n>` | Увімкнути обрізання блокчейну (зменшити використання диска) |
| `-dbcache=<n>` | Встановити розмір кешу бази даних у МБ (за замовчуванням: 450) |
| `-maxconnections=<n>` | Максимальна кількість з'єднань з вузлами (за замовчуванням: 128) |
| `-listen` | Приймати вхідні з'єднання (за замовчуванням: 1) |
| `-server` | Увімкнути RPC-сервер |

### Запуск через робочий стіл (GUI)

#### Windows
- Двічі клацніть на файлі `taler-qt.exe` або використовуйте ярлик на робочому столі

#### macOS
- Перейдіть до розпакованої директорії
- Двічі клацніть на `taler-qt` (якщо встановлено в `/usr/local/bin`, запустіть з терміналу: `taler-qt`)
- Або запустіть з розпакованої директорії: `./bin/taler-qt`

#### Linux
- Використовуйте засіб запуску застосунків для пошуку "TALER"
- Або запустіть з терміналу: `taler-qt`
- Або з розпакованої директорії: `./bin/taler-qt`

## Початкове налаштування

### Початкова синхронізація блокчейну

При першому запуску гаманця TALER він почне синхронізацію з блокчейном:

1. **Екран заставки**: Застосунок відобразить логотип TALER під час завантаження
2. **Повідомлення про синхронізацію**: З'явиться модальне вікно з прогресом синхронізації
3. **Індикатор прогресу**: Показує відсоток завантаженого блокчейну
4. **Оцінка часу**: Відображає залишковий передбачуваний час

**Примітка**: Початкова синхронізація може зайняти кілька годин залежно від швидкості вашого інтернет-з'єднання. Гаманець повинен завантажити всю історію блокчейну.

### Розуміння процесу синхронізації

- **Висота блоку**: Поточний блок проти останнього мережевого блоку
- **Швидкість завантаження**: Пропускна здатність мережі у КБ/с або МБ/с
- **Спочатку заголовки**: Завантажує заголовки блоків перед повними блоками для швидшої початкової синхронізації
- **Фонова синхронізація**: Ви можете згорнути гаманець, поки синхронізація продовжується

**Порада**: Для швидшої синхронізації переконайтеся, що ваша система дозволяє TALER через брандмауер і має достатню кількість з'єднань з вузлами (перевірте рядок стану).

## Огляд інтерфейсу гаманця

Після синхронізації інтерфейс гаманця TALER складається з кількох основних розділів:

### Верхні вкладки навігації

1. **Огляд**: Панель керування, що показує баланси та останні транзакції
2. **Надіслати**: Інтерфейс для надсилання монет TALER
3. **Отримати**: Генерація адрес для отримання монет
4. **Транзакції**: Повна історія транзакцій
5. **Майнінг**: Увімкнення/налаштування PoW майнінгу
6. **Стейкінг**: Налаштування PoS стейкінгу (доступно після синхронізації)

### Рядок стану (внизу)

- **Статус з'єднання**: Кількість активних з'єднань з вузлами
- **Кількість блоків**: Поточна висота синхронізованого блоку
- **Статус мережі**: Хешрейт мережі та складність
- **Статус гаманця**: Статус блокування/розблокування (зашифровані гаманці)

### Головна панель інструментів

- **Файл**: Резервне копіювання гаманця, експорт ключів, вихід
- **Налаштування**: Налаштування параметрів гаманця, мережевих параметрів
- **Інструменти**: Консоль налагодження, підпис/перевірка повідомлень, інформація про вузли
- **Довідка**: Діалог "Про програму", параметри командного рядка

## Створення вашого першого гаманця

Застосунок TALER створює гаманець автоматично при першому запуску. Файл гаманця зберігається у вашій директорії даних:

### Розташування гаманця за замовчуванням

- **Windows**: `%APPDATA%\TALER\wallet.dat`
- **macOS**: `~/Taler/wallet.dat`
- **Linux**: `~/.taler/wallet.dat`
