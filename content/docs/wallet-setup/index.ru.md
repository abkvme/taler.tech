Это руководство проведет вас через процесс установки и настройки кошелька TALER в вашей системе.

## Установка

### Установка в Windows

1. **Распакуйте архив**:
   - Щелкните правой кнопкой мыши на `taler-v0.18.44.7-win64.zip`
   - Выберите "Извлечь все..."
   - Выберите папку назначения (например, `C:\Program Files\TALER`)

2. **Найдите исполняемый файл**:
   - Перейдите в распакованную папку
   - Найдите `taler-qt.exe` (приложение GUI-кошелька)

3. **Опционально - Создайте ярлык на рабочем столе**:
   - Щелкните правой кнопкой мыши на `taler-qt.exe`
   - Выберите "Отправить" → "Рабочий стол (создать ярлык)"

### Установка в macOS

1. **Распакуйте архив**:
   ```bash
   tar -xzvf taler-v0.18.44.7-macos-arm64.tar.gz
   ```

2. **Переместите в Applications** (опционально):
   ```bash
   sudo mv taler-v0.18.44.7/bin/* /usr/local/bin/
   ```

3. **Предоставьте права на выполнение**:
   ```bash
   chmod +x /usr/local/bin/taler-qt
   ```

### Установка в Linux

1. **Распакуйте архив**:
   ```bash
   tar -xzvf taler-v0.18.44.7-linux-x64.tar.gz
   cd taler-v0.18.44.7
   ```

2. **Установите в систему** (опционально):
   ```bash
   sudo install -m 0755 -o root -g root -t /usr/local/bin bin/*
   ```

3. **Или запустите из распакованной директории**:
   ```bash
   ./bin/taler-qt
   ```

## Запуск кошелька TALER

### Запуск через командную строку

GUI-кошелек TALER можно запустить из командной строки с различными опциями.

#### Базовый запуск

**Windows**:
```cmd
taler-qt.exe
```

**macOS / Linux**:
```bash
taler-qt
```

#### Запуск с пользовательской директорией данных

Укажите пользовательское расположение для данных блокчейна:

```bash
taler-qt -datadir=/path/to/custom/directory
```

#### Запуск в тестовой сети

Для целей тестирования запустите кошелек в тестовой сети:

```bash
taler-qt -testnet
```

#### Общие параметры командной строки

| Опция | Описание |
|--------|-------------|
| `-datadir=<dir>` | Указать расположение директории данных |
| `-conf=<file>` | Указать файл конфигурации |
| `-testnet` | Использовать тестовую сеть |
| `-prune=<n>` | Включить обрезку блокчейна (уменьшить использование диска) |
| `-dbcache=<n>` | Установить размер кеша базы данных в МБ (по умолчанию: 450) |
| `-maxconnections=<n>` | Максимальное количество соединений с узлами (по умолчанию: 128) |
| `-listen` | Принимать входящие соединения (по умолчанию: 1) |
| `-server` | Включить RPC-сервер |

### Запуск через рабочий стол (GUI)

#### Windows
- Дважды щелкните на файле `taler-qt.exe` или используйте ярлык на рабочем столе

#### macOS
- Перейдите в распакованную директорию
- Дважды щелкните на `taler-qt` (если установлено в `/usr/local/bin`, запустите из терминала: `taler-qt`)
- Или запустите из распакованной директории: `./bin/taler-qt`

#### Linux
- Используйте средство запуска приложений для поиска "TALER"
- Или запустите из терминала: `taler-qt`
- Или из распакованной директории: `./bin/taler-qt`

## Первоначальная настройка

### Начальная синхронизация блокчейна

При первом запуске кошелька TALER он начнет синхронизацию с блокчейном:

1. **Экран заставки**: Приложение отобразит логотип TALER во время загрузки
2. **Уведомление о синхронизации**: Появится модальное окно с прогрессом синхронизации
3. **Индикатор прогресса**: Показывает процент загруженного блокчейна
4. **Оценка времени**: Отображает оставшееся предполагаемое время

**Примечание**: Начальная синхронизация может занять несколько часов в зависимости от скорости вашего интернет-соединения. Кошелек должен загрузить всю историю блокчейна.

### Понимание процесса синхронизации

- **Высота блока**: Текущий блок против последнего сетевого блока
- **Скорость загрузки**: Пропускная способность сети в КБ/с или МБ/с
- **Сначала заголовки**: Загружает заголовки блоков перед полными блоками для более быстрой начальной синхронизации
- **Фоновая синхронизация**: Вы можете свернуть кошелек, пока синхронизация продолжается

**Совет**: Для более быстрой синхронизации убедитесь, что ваша система разрешает TALER через брандмауэр и имеет достаточное количество соединений с узлами (проверьте строку состояния).

## Обзор интерфейса кошелька

После синхронизации интерфейс кошелька TALER состоит из нескольких основных разделов:

### Верхние вкладки навигации

1. **Обзор**: Панель управления, показывающая балансы и последние транзакции
2. **Отправить**: Интерфейс для отправки монет TALER
3. **Получить**: Генерация адресов для получения монет
4. **Транзакции**: Полная история транзакций
5. **Майнинг**: Включение/настройка PoW майнинга
6. **Стейкинг**: Настройка PoS стейкинга (доступно после синхронизации)

### Строка состояния (внизу)

- **Статус соединения**: Количество активных соединений с узлами
- **Количество блоков**: Текущая высота синхронизированного блока
- **Статус сети**: Хешрейт сети и сложность
- **Статус кошелька**: Статус блокировки/разблокировки (зашифрованные кошельки)

### Главная панель инструментов

- **Файл**: Резервное копирование кошелька, экспорт ключей, выход
- **Настройки**: Настройка параметров кошелька, сетевых параметров
- **Инструменты**: Консоль отладки, подпись/проверка сообщений, информация об узлах
- **Справка**: Диалог "О программе", параметры командной строки

## Создание вашего первого кошелька

Приложение TALER создает кошелек автоматически при первом запуске. Файл кошелька хранится в вашей директории данных:

### Расположение кошелька по умолчанию

- **Windows**: `%APPDATA%\TALER\wallet.dat`
- **macOS**: `~/Taler/wallet.dat`
- **Linux**: `~/.taler/wallet.dat`
